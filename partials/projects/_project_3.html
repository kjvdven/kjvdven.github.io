Batavia Stad BMember

A customer loyalty program for Batavia Stad.

To encourage return visits to Batavia Stad Outlet Shopping, a customer loyalty program was started. To give the best user experience the system was tied into the excisting website and newsletter.

The BMember experience consistsed of:

  • A kiosk in the main buildings lobby;
  • a members section on the Batavia Stad Website;
  • an api for the iPhone App; and
  • a registry system for in the shops.
